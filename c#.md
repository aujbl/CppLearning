# C#

1. 一个 C# 程序主要包括以下部分：

    ```c#
    using System;
    namespace HelloWorldApplication                     // 命名空间声明（Namespace declaration）
    {
    class HelloWorld                                    // 一个 class
    {
                                                        // Class 方法
                                                        // Class 属性
        static void Main(string[] args)                 // 一个 Main 方法      
                                                        // 语句（Statements）& 表达式（Expressions）
        {
            /* 我的第一个 C# 程序*/                      // 注释
            Console.WriteLine("Hello World");
            Console.ReadKey();
        }
    }
    }                                                   // C# 文件的后缀为 .cs
    ```
